Hablemos de Van Gogh


Imagen tomada de este sitio web.

¿Qué es esto?

Un sitio web dedicado a construir una comunidad alrededor de las obras del artista número uno del postimpresionismo, el Holandés Vincent Willem Van Gogh. Con el fin de apoyar, compartir, discutir y teorízar con respecto a su obra y vida.

¿Quienes somos?

Al igual que muchas personas alrededor del mundo, deseamos conectar con estas a partir de este gusto particular por las pinturas de este asombroso artista. Cabe aclarar que de ninguna manera este sitio no se encuentra directamente relacionado con el autor ni mucho menos la administración de las obras de Van Gogh.
